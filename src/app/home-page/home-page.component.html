Тестовое задание на должность javascript developer

Разработан прототип приложения системы ведения заявок для логистов в авто и грузоперевозках в виде web-приложения

Проект полностью рабочий, т.е. его можно собрать и запустить из исходного кода и весь основной функционал реализован.

Также реализованы некоторые дополнительные задания, такие как: поиск по таблице заявок (по названию компании), а также фильтрация таблицы по количеству одновременно находящихся на ее странице элементов(по своей инизиатие добавлена пагинация)

Прототип включает в себя таблицу с заявками, возможность просмотра, редактирования и удаления заявок, создание новых заявок.

В заявке есть следующие данные:
  • ● Номер заявки
  • ● Дата и время получения заявки от клиента
  • ● Название фирмы клиента
  • ● ФИО перевозчика
  • ● Контактный телефон перевозчика
  • ● Комментарии
  • ● ATI код сети перевозчика (кликабельно, переход на сайт)

Для хранения данных о заявках использована база данных на Firebase, что позволяет полноценно взаимодействовать с приложением и сохранять все изменения даже при обновлении страницы.
